# Corporate B2B Landing Page — Quick Start Guide

## Customize in 5 minutes

Open `corporate-template.html` in any text editor and change these CSS variables:

```css
:root {
  --navy:  #0f1c3a;   /* ← dark background */
  --gold:  #c9a84c;   /* ← accent color */
  --red:   #c0392b;   /* ← secondary accent */
}
```

### Replace content:
- `MM SecurityService` → your company name
- `Milano e provincia` → your service area
- Phone/email in the contact section

### Add your logo:
Replace the `.nav-logo` SVG with your own or an `<img>` tag.

## Deploy
Single HTML file — drop into Netlify, GitHub Pages, any web host.
