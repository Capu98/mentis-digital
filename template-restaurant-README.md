# Restaurant Landing Page — Quick Start Guide

## Customize in 5 minutes

Open `restaurant-template.html` in any text editor and change these at the top of the file:

```css
:root {
  --teal:  #00d4b4;   /* ← your main accent color */
  --bg:    #07090f;   /* ← background (dark recommended) */
}
```

### Replace content:
- `BotReady` → your restaurant name
- `risponde alle prenotazioni` → your main service/tagline
- WhatsApp link: search `wa.me/` and replace with your number

### Add your logo:
Replace the `.nav-logo` text with an `<img>` tag.

## Deploy
Drop the single HTML file into Netlify, GitHub Pages, or any web host.
No server needed — pure static HTML.
